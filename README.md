# router-crash-button
Router Crash Button

## Requirement
Ruby 2.3.0
libpcap-dev

```
$ sudo apt install libpcap-dev
```

## Setup

```
$ git clone https://github.com/kooshin/router-crash-button
$ bundle
```

## Execute

```
$ sudo ruby crash_button.rb
or
$ rbenv sudo ruby crash_button.rb
Waiting for Crash Button press...
```

Push "Amazon Dash Button"
